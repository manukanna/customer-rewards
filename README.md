# Customer Rewards
 This is a React-based application that calculates and displays customer reward points based on their transaction history. It provides an interactive UI with search functionality and detailed customer insights.

# Project Overview
- Intially User has to login with email and password
- User Login Successfully Routing to customer-rewrds
- Displaying user rewards Points Cards
- Calculate reward points based on transaction amount
- Display total transaction amount per customer
- Show total reward points per customer
- Search customers by name
- Responsive UI using Bootstrap 5
- Click on card to view detailed customer info (Modal)
- Optimized performance using React hooks (useMemo)


# Techstack 
- React.js
- Bootstrap 5
- React Router DOM
- JavaScript (ES6+)
- HTML5 & CSS3

# Key Components
- CustomerLogin → Customer Login
